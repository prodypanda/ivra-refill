## 2024-05-18 - Avoid O(N) context lookups inside loops\n**Learning:** In Flutter, calling `Localizations.localeOf(context)` or inherited widget lookups inside a tight loop or `.where` function is O(N) because the widget lookup happens N times, allocating resources each iteration.\n**Action:** Pre-compute the `localeOf` or any `toLowerCase()` operations before mapping over or filtering lists.
